Pothole Detector Using IoT

It's a project that uses an IoT gadget mounted on a Moped, bike, or car to identify potholes on the road. Which will locate the pothole and display it on Maps. ELK(Elastic, Logstash, Kibana) stack was used in this project. For storing the data we used Elasticsearch and Logstash on IoT device for persistant queue of data, Kibana for data visualization like Potholes on Maps, Car or Bike information like cars engine status and all... Also monitoring for IoT Device is added.
IoT Device: Raspberry Pi
